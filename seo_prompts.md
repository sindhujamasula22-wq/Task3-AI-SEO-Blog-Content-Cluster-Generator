# Task 3 — AI SEO Prompt System
## AI SEO Blog & Content Cluster Generator

---

## PROMPT 1: Keyword Research

```
Act as an SEO expert. I have a business in the niche: [NICHE] located in [CITY].

Generate a keyword research table with 10 keywords including:
- Primary keyword (1 main target)
- 9 secondary/LSI keywords

For each keyword provide:
1. Keyword phrase
2. Estimated monthly search volume
3. Keyword difficulty (Easy / Medium / Hard)
4. Search intent (Informational / Transactional / Commercial / Navigational)
5. Recommended blog type (Pillar / Cluster)

Format the output as a table.
```

---

## PROMPT 2: Content Cluster Map

```
Act as a content strategist. Based on this primary keyword: [PRIMARY KEYWORD]
for a [NICHE] business in [CITY], create a content cluster map.

Include:
- 1 Pillar Post title (main topic, broad, 2000+ words)
- 5 Cluster Post titles (subtopics that link back to pillar)
- For each post: suggested H1 title, target keyword, search intent, word count

Make all titles optimized for local SEO using [CITY] + [SERVICE] format where relevant.
```

---

## PROMPT 3: Blog Outline (H1–H3 Structure)

```
Act as an SEO content writer. Write a detailed blog outline for this topic:
Title: [BLOG TITLE]
Target Keyword: [KEYWORD]
Location: [CITY]
Search Intent: [INTENT]
Word Count Target: [WORD COUNT]

Include:
- SEO Meta Title (under 60 characters)
- Meta Description (under 160 characters)
- H1 heading
- 5–7 H2 sections
- 2–3 H3 sub-sections under each H2
- CTA suggestion at the end
- 3 internal linking suggestions (titles of cluster posts to link to)
```

---

## PROMPT 4: Long-Form Blog Body

```
Act as an SEO blog writer. Write a complete, long-form blog post using this outline:

[PASTE OUTLINE FROM PROMPT 3]

Requirements:
- Word count: [TARGET WORDS]
- Include the target keyword [KEYWORD] naturally 6–8 times
- Use [CITY] in the introduction, at least 2 H2s, and conclusion
- Write in a helpful, conversational tone (Grade 8 reading level)
- Add a clear CTA (Call to Action) at the end
- Include 1 FAQ section with 3 questions
- Do NOT keyword stuff — keep it natural and reader-first
```

---

## PROMPT 5: Meta Tags Generator

```
Act as an on-page SEO specialist. Write optimized meta tags for this blog post:

Title: [BLOG TITLE]
Target Keyword: [KEYWORD]
City: [CITY]
Blog Summary: [2-LINE SUMMARY]

Generate:
1. SEO Meta Title — include keyword + city, under 60 characters
2. Meta Description — compelling, includes keyword, under 160 characters
3. URL Slug — lowercase, hyphenated, keyword-first
4. Focus Keyphrase
5. 5 related tags/categories
```

---

## PROMPT 6: Internal Linking & Content Cluster Connector

```
I have written these blog posts for my [NICHE] website:

1. [PILLAR POST TITLE]
2. [CLUSTER 1 TITLE]
3. [CLUSTER 2 TITLE]
4. [CLUSTER 3 TITLE]
5. [CLUSTER 4 TITLE]
6. [CLUSTER 5 TITLE]

For each post, suggest:
- 2 internal links TO this post (from which other posts)
- 2 internal links FROM this post (to which other posts)
- Anchor text to use for each link

Format as a linking map table.
```

---

## PROMPT 7: Local SEO Adaptation

```
Take this blog post section and rewrite it to be optimized for local SEO.

Original text: [PASTE SECTION]

Requirements:
- Naturally include the city name [CITY] 2–3 times
- Add one local reference (e.g. area name, landmark, or local context)
- Keep the [KEYWORD] in the first 100 words
- Maintain the same tone and length
- Do NOT make it sound forced or spammy
```
